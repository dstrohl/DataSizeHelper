Data Size Calculator¶

This object handles converting between one unit to another unit.

Notes:
  • All numbers entered will be converted to decimals and all return values are of type decimal.Decimal
  • Unit names passed must be correct standard abbreviations for the units, (use the DataUnitManager to normalize and validate unit names if needed).
class data_size_helper.DataSizeHelper(value=0, unit=None, suffix=None, default_unit=None, storage_unit='b', unitset=None, unit_manager=None, calculator=None)[source]¶
Parameters:
  • value – The value of the object
  • unit – The unit for the object
  • suffix – Any suffix (‘/s’ for example)
  • default_unit – If just a number is passed, what unit do we assume it is
  • storage_unit – Normally everything is stored in bits, however when working with very large numbers it may be easier to store data in a different unit. if all units will be the same, this may reduce calculation time as well.
  • unitset – the unitset may need to be passed if normalization is required and the system cannot automatically determine it. (primarily if working with ‘B’ or ‘b’).
  • unit_manager – if desired, allows overriding the unit manager which handles unit name detection and normalization
  • calculator – If desired, allows overriding the unit calculator which handles converting between units.

Examples (note, returns are shown as literal int or float, however are actually of type Decimal).:

>>> from DataSizeHelper import data_size_calculator,
>>> dsc = data_size_calculator
>>> dsc(1000, 'Kb', 'Mb')
(1, 'Mb')
>>> dsc(1, 'b', 'Kib')
(1024, 'Kib')
>>> data_size_calculator(1000, 'Mb', dsc.DEC_BIT)
(1, 'Gb')
